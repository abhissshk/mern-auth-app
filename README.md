# 🔐 MERN Authentication App

A full-stack authentication system built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This project includes user signup, login, JWT authentication, and protected routes.

---

## 🚀 Features

- 🔑 User Signup & Login
- 🔐 Password Hashing using bcrypt
- 🪪 JWT Authentication
- 🔒 Protected API Routes
- 💾 MongoDB Database Integration
- ⚛️ React Frontend with Form Handling
- 🔔 Toast Notifications

---

## Tech Stack

### Frontend:
- React.js
- React Router
- React Toastify

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT (jsonwebtoken)
- bcrypt
